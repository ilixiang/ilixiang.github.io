2020-08-08
python

记下这篇的主要原因是平时python的使用极少，因而常在学习和忘记的状态反复横跳，所以记下一些自己常忘记的语法和操作。

def func():

yield是python中的关键字，如果函数中使用了它，那么调用函数得到的就是一个生成器而并非返回值

生成器可以通过调用next()从上次yield返回的地发继续执行

生成器的特点有：

  1. 惰性求值，仅在每次调用next()的时候求取下一个值，内存友好
  2. 能够无限返回，这一点是普通函数做不到的

def decorateFunc(func):
    def innerFunc(*args, **kwargs):
        print("Before execute")
        func(*args, **kwargs)
        print("After execute")
    return innerFunc

@decorateFunc
def func():
    print("Hello world")
func()
'''输出

Hello world
After execute
'''

python的装饰器是一种函数式编程的应用，通过将函数作为变量、参数、返回值，从而利用已有函数生成新的函数。

def decorateFunc(a, b):
    def innerDecorate(func):
        def innerFunc(*args, **kwargs):
            print("This is innerFunc")
            print("a is {}, b is {}".format(a, b))
            func(*args, **kwargs)
        return innerFunc
    return innerDecorate

@decorateFunc(1, 2)
def func(*args):
    for arg in args:
        print(arg)

func(1, 2, 3, 4)

'''输出
This is innerFunc
a is 1, b is 2
1
2
3
4
'''

上面这个装饰器更为复杂，它拥有参数，因此实现需要3层的函数嵌套。

  • 第一层的def decorateFunc(a, b)的作用是返回一个无参数的装饰器，就如同前面那个例子一样

  • 第二层的def innerDecorate(func)的作用是传入被装饰的函数，返回装饰后的函数

  • 第三层的def innerFunc(*args, **kwargs)用于定义装饰后的函数

class R:
    def __init__(self, s):
        self.s = s
        print("call __init__")
    def __enter__(self):
        print("call __enter__")
        return self
    def __exit__(self, type, value, traceback):
        print("call __exit__")
with R("hello world") as r:
    print(r.s)
'''输出
call __init__
call __enter__
hello world
call __exit__
'''

python中的with关键字所使用的语法类似与Java中的try-with-resource，能够不用去写finally调用相应资源的close()方法

python中的with func() as f调用顺序为：

  1. 调用func()获取返回对象
  2. 调用返回对象的__enter__()方法获取相应的返回值并赋值给f
  3. 使用f进行相应的操作
  4. 调用func()返回对象的__exit__销毁资源

2 基本操作

2.1 IO

2.1.1 输入输出

input()是python的输出函数，功能类似与Java中的Scanner，能够按行读取来自控制台的输入

上面是输出函数print()方法的完整定义，因而通过print()方法能够指定分隔符、终止符、输出文件等参数，从而获得自己想要的输出格式

2.1.2 文件操作

文件操作使用的函数是open()，完整定义为：

1

open(file, mode='r', buffering=-1, encoding=None, errors=None, newline=None, closefd=True, opener=None)

其中比较关键的是mode参数，仅针对文本文件有6中选择：

  1. r，只读方式，如果文件不存在则抛出异常，属于默认模式
  2. r+，读写方式，写入位置为文件的起始位置，文件不存在则会抛出遗产
  3. w，只写方式，若文件长度不为0则将之截断为0，文件不存在则创建
  4. w+，读写方式，若文件长度不为0则将之截断为0，文件不存在则创建
  5. a，只写方式，以追加方式写入，文件不存在则创建
  6. a+，读写方式，以追加方式写入，文件不存在则创建

2.2 集合

2.2.1 集合构造

python中的集合创建大多数都不需要调用构造器：

  1. dict，字典类型，构造方法是d = {}，在Java中对应Map，存储键值对
  2. list，列表类型，构造方法是l = []，在Java中对应List
  3. tuple，元组类型，构造方法是t = ()，但元组一经初始化就无法添加/删除/更换元素
  4. set，集合类型，构造方法是s = set()，由于set和dict都是使用{}表示，只不过一个存储关联数据，一个仅存单个数据，因此初始化一个空set需要调用构造器
